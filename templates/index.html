MaxPanel, stats service

How to use:

  • Register user here: /api/v1/user/register/ with payload: 
    {"email": "...", "password": "minimum 4 symbols"}
  • Login user here: /api/v1/user/login/ with payload: 
    {"email": "...", "password": "..."}
  • Register your application here: /api/v1/register/ with POST: 
    {"app_name": "test"}
  • Save data to route: /api/v1/stats/ via POST-request with application API key in header: 
    API-MaxPanel: ...
  • You can send only primitive types: integer, strings etc
  • For retrieve stats: /api/v1/export/ with API key in header 
    API-MaxPanel: ...
    Available query arugments is:
    • from_date
    • to_date
